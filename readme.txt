Program ini akan melakukan auto checkout di website shopee  
Semua akan di lakukan secara acak barang yang mempunyai pilihan seperti 'warna' akan di pilih secara acak
tetapi hanya work di 1 pilihan tidak work di 2 pilihan
Jangan lupa mematikan OTP di akun anda sebelum menjalankan bot ini


Cara Pakai:
- Pasikan kalian menggunakan chrome versi 87
- Masukan akun shopee ke login.txt
- Masukan link shopee yang ingin di checkout ke url.txt
